# SCT_DS_4 - Business Insights Report

## Project Overview
This project focuses on analyzing a marketing campaign dataset to identify customer behavior patterns and generate actionable business insights.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze customer spending behavior
- Identify business opportunities
- Generate data-driven recommendations

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Tasks Performed
- Data Loading
- Missing Value Handling
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization
- Business Insight Generation

## Key Findings
- Higher income customers tend to spend more.
- Spending behavior varies across education levels.
- Product categories contribute differently to overall spending.
